Libraries

  • moment.js (http://momentjs.com/docs/) - Great library for time manipulation / formatting in general, but also has great i18n support.
  • angular-gettext (https://angular-gettext.rocketeer.be/dev-guide/) - String extraction / translation in language packs for angular using angular modules. Also supports pluralization.
  • angular-i18n (https://code.angularjs.org/1.3.14/docs/guide/i18n) - Core Angular module for adding i18n support to number, date, and currency angular filters.
  • ECMA International API (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl) - Core Javascript API for handling locale-aware string sorting (among other things).
  • jsTimezoneDetect (https://bitbucket.org/pellepim/jstimezonedetect) - Automatic client-side detection of an application user's timezone
  • angular-dynamic-locale (http://lgalfaso.github.io/angular-dynamic-locale/) - Dynamic loading of appropriate angular-i18n language pack
